# GitHubReposApp

Git Hub Repos App は**GitHubAPI**を使い、ユーザーを検索し、リポジトリを探すことができるアプリです。  
**RxSwift** を使いMVVMアーキテクチャパターンによる実装を行いました。  
API通信や非同期処理や
RxDataSourceを使うことでTablevViewと表示データのバインド処理を実装しました。
SearchUser画面ではサーチバーからユーザーを検索し検索結果を表示します。
